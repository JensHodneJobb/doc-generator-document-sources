# Control systems, HMI and networks

## General
- HMH delivers complete, integrated control and monitoring systems for drilling and well intervention.
- HMH's control systems are designed to ensure safe, efficient and reliable working conditions and operations
- The overall functionality is to manage the flow of control and information within the drilling system and with the operator.
- Control systems from HMH are open for interconnection internally and with other vendors
- Rig Safety and Automation System (SAS) is not part of the HMH offering.
- Compliance to DNV-OS-D203 ISDS or ABS ISQM can be delivered as an option
- A comprehensive service interface (HMI) for expected maintenance and fault finding is included for all control systems.

## Hardware
- HMI systems, data logging, analysis and systems for advanced calculations are running on industrial Windows compatible computers.
- Programmable Logic Controllers (PLC) are of the type Siemens Simatic S7, conforming to IEC 61131-3
- I/O modules are connected to CPU's through fieldbus, enabling cost efficient control system upgrades in the future.

## Network Design
- An installation contains two main Ethernet networks
- The PLC network interconnects PLCs. This includes connecting equipment control systems to operator chair and anti-collision.
- The DrillView network connects HMI system and equipment control systems.
- Important network signals have redundant communication lines.
- I/O fieldbuses are based on Profibus and/or Profinet according to IEC 61158
- For all network / fieldbus interfaces where HMH do not have control of cable routing, fiber optical cabling must be used.

## Operator Environment and Human Machine Interface (HMI)
- DrillView is an open HMI system for distribution, monitoring and control of drilling and process data (real time and stored)
- The DrillView alarm system is in compliance with the principles of IEC 62682 and EEMUA 191.
- Different operator environments can be provided:
- DrillView clients – industrial or office computers with high definition displays
- Operator chairs - with touch panels, joysticks, and mouse for PC interface
- Panels - radio panels, hydraulic panels, electrical panels, touch panels
- The operator environment supports different modes of operation, from purely manual to highly automated and robotic control.

## Open Interfaces
- HMH systems can interface a wide range of Variable Frequency Drives (VFD), motor starters and information management systems (IMS) upon request.
- HMH HMI and control system can interface any 3rd party equipment like wireline, cement systems or other equipment.
- HMH supplied equipment can interface any 3rd party HMI system upon request

## Built in Capabilities
- HMH equipment and process control systems have built in capabilities for connection to our comprehensive portfolio of rig automation, advanced support systems and services.

## Cyber security

HMH's cyber security program is based on IEC-62443-3-3 and  IEC 62443-2-4.

Security Level 1 is standard for our cyber security product offering. Higher security levels can be delivered upon request.
