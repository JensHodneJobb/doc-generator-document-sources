# API Monogram

The following equipment may be supplied with API monogram:

Rotary tables API 7K-0551

Mud pump components API 7K-0551

Drawwork components API 7K-0551

Power swivels (Top drive) PSL 1 API 8C-0351

Travelling blocks PSL 1 API 8C-0351

The implementation of API monogram license for other product types is continuously evaluated by HMH, based on market demand.Current monogram certifications are published at the "API composite list" under company names MHWirth or HMH.
