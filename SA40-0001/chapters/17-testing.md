# Testing

Within the scope of this standard HMH equipment is typically subjected to the following tests:
- Manufacturing tests and inspection of components and sub-assemblies
- Assembly test and inspection. (Various early functional trials in workshop.)
- Mechanical completion (MC inspection)
- Internal acceptance test, including mechanical, electric, fluid power, control system, safety functions, MC check out and load tests as applicable for lifting equipment
- Factory acceptance test

If the equipment cannot be fully functional tested before final installation, test plans are developed for each sub system to be tested as far as practical in the workshop.

HMH will supply procedures for installation and testing on board, and can also provide test personnel upon request(Test execution/assistance by HMH during/after installation is optional, and must be specified in contract/purchase order).
